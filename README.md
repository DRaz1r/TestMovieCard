# 电影卡片--MovieCard
## APP名
**爱奇小艺**
* 我们希望最终能对标国内顶尖的视频网站，比如爱奇艺，并向着这个目标努力
* 同时我们希望带给用户便捷的体验，于是联想到本次课程，联想到华为，我们将‘小艺’的智慧加入其中
